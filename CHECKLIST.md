## 🚀 Getting Started (3 Steps)

### 1. Install
```bash

```

### 2. Run
```bash
python server.py              # Terminal 1
```

### 3. Access
- Frontend: http://localhost:3000
- Admin: http://localhost:3000/admin/login
- API: http://localhost:5000/api
