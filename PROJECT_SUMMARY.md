## 🔌 API Endpoints

### Public Endpoints (15)
- GET `/api/info` - Organization info
- GET `/api/events` - All events
- GET `/api/athletes` - All athletes
- GET `/api/news` - News articles
- GET `/api/tournaments` - Tournaments
- GET `/api/rankings` - Rankings
- GET `/api/gallery` - Gallery images
- GET `/api/training-programs` - Training
- GET `/api/schedule` - Weekly schedule
- GET `/api/statistics` - Basic stats
- GET `/api/stats` - Detailed stats
- GET `/api/announcements` - Announcements
- GET `/api/coaches` - Coaches list
- GET `/api/members` - Members list

npm run build && git push production main
